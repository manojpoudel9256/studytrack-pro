# StudyTrack Pro 🚀

StudyTrack Proは、学生や専門家が学習時間を追跡し、生産性を分析し、ゲーミフィケーションを通じてモチベーションを維持するために設計された、最新のフルスタックWebアプリケーションです。

## ✨ 機能 (Features)

### 📊 ダッシュボード & 分析
- **視覚的分析**: 学習傾向を視覚化するためのインタラクティブな棒グラフと円グラフ。
- **統計概要**: 総学習時間、完了したタスク、現在の継続記録（ストリーク）の概要。
- **アクティビティ履歴**: フィルタリングオプション付きのすべての学習セッション詳細ログ。

### ⏱️ 集中ツール
- **内蔵タイマー**: ライブ学習セッションを追跡するための専用フォーカスタイマー。
- **リアルタイムログ**: タイマーが停止すると自動的に記録を作成します。

### 🏆 ゲーミフィケーション
- **リーダーボード**: 学習時間（XP）に基づいて他のユーザーと競います。
- **ランキングシステム**: 勉強すればするほど、ランクやバッジを獲得できます。
- **ダイナミックスコアリング**: 勉強1分 = 10 XP。

### 👤 ユーザープロフィール
- **パーソナライズされたプロフィール**: レベル、参加日、統計を表示します。
- **カスタムアバター**: 独自のプロフィール写真をアップロード（切り抜きとサイズ変更をサポート）。
- **詳細編集**: 名前、メールアドレス、パスワードを簡単に更新できます。

### 🌐 国際化 (i18n)
- **多言語サポート**: **英語**と**日本語**に完全対応。
- **統合トグル**: ログイン/登録画面から言語をシームレスに切り替え可能。
- **設定の保存**: 言語設定を自動的に記憶します。

### ✨ "Golden Light" テーマ & UI
- **プレミアムな美学**: 柔らかなクリーム色の背景を持つ、温かみのある高貴な**ゴールド/アンバー**テーマ。
- **ライブ背景**: ログイン＆登録ページには、浮遊する幾何学的な粒子を伴う**ダイナミックな動きのあるグラデーション**を採用。
- **モダンなインタラクション**: ボタンには、高いコントラストとモダンな雰囲気を持つスタイリッシュな**ブルー・パープル**のグラデーションを採用。

### 💬 モチベーション引用句
- **毎日のインスピレーション**: ダッシュボードに新しいモチベーション引用句を表示します。
- **ハイブリッドシステム**: ライブAPI（英語）から取得するか、厳選されたオフラインライブラリ（日本語）を使用します。

## 🛠️ 技術スタック

- **フロントエンド**: React.js (Vite), Tailwind CSS v4, Framer Motion (アニメーション), Recharts (チャート), Lucide React (アイコン).
- **国際化**: i18next, react-i18next, i18next-browser-languagedetector.
- **バックエンド**: Node.js, Express.js.
- **データベース**: MySQL.
- **認証**: JWT (JSON Web Tokens), BCrypt (パスワードハッシュ化).
- **ファイル処理**: Multer (画像アップロード).

## 🚀 始め方 (Getting Started)

以下の手順に従って、プロジェクトをローカルで実行してください。

### 前提条件
- Node.js がインストールされていること。
- MySQL がインストールされ、実行されていること（例：XAMPP経由）。

### 1. データベースのセットアップ
1. MySQLクライアント（例：phpMyAdmin）を開きます。
2. `studytrack_db` という名前の新しいデータベースを作成します。
3. `database/schema.sql` ファイルをインポートしてテーブルを作成します。

### 2. バックエンドのセットアップ
```bash
cd backend
npm install
# DB認証情報とJWT_SECRETを含む .env ファイルを作成してください
node server.js
```

### 3. フロントエンドのセットアップ
```bash
cd frontend
npm install
npm run dev
```

アプリは `http://localhost:5173` で起動するはずです。

## 📂 プロジェクト構成

```
studytrack-pro/
├── backend/            # Express API & データベースロジック
│   ├── controllers/    # API コントローラー
│   ├── routes/         # API ルート
│   └── uploads/        # ユーザーがアップロードした画像
├── frontend/           # React アプリケーション
│   ├── src/
│   │   ├── components/ # 再利用可能なUIコンポーネント
│   │   ├── pages/      # メインアプリケーションページ
│   │   └── api/        # API 統合
│   └── index.css       # Tailwind インポート
├── database/           # SQL スキーマ
├── design/             # アーキテクチャと計画ドキュメント
└── screenshots/        # プロジェクト画像
```

## 🔒 セキュリティ
- パスワードは **bcrypt** を使用してハッシュ化されます。
- APIエンドポイントは **JWT** を使用して保護されています。
- ファイルアップロードはセキュリティのために検証およびリネームされます。
